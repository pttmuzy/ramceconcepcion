Teemr

Teemr is a small JavaScript theme library that allows you to dynamically apply specific sets of css properties on your html elements based on JSON templates, thus, giving your web site or web app the capability to have multiple themes. Especially useful when managing color schemes. Try it below:


Speck

Speck is an HTML-based semi-customizable mini particle generator and animation. Spice up your website header with moving particles. Look at this website's header to see how it works!

Workr

Workr is a JavaScript helper library that allows you to use the native Web Worker API more intuitively and creatively. Workr also enables its user to pass function references to a web worker instead of creating separate .js files -- effectively making any valid method from your codebase run on a separate thread. (Demo not yet available)

Alertr

Alertr is a customizable JavaScript alert UI component similar to Windows 10 notifications. You can show stylized notification pop ups anywhere on the screen. (Demo not yet available)
