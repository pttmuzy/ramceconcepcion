Themify

Themify is a small JavaScript theme library that allows you to dynamically apply specific sets of css properties on your html elements based on JSON templates, thus, giving your web site or web app the capability to have multiple themes. Workr

Workr is a JavaScript helper library that allows you to use the native Web Worker API more intuitively and creatively. Workr also enables its user to pass function references to a web worker instead of creating separate .js files. (Demo not yet available)



Alertr

Alertr is a customizable JavaScript alert and notification UI component similar to Windows 10 notifications. You can show stylized notification pop ups anywhere on the four corners of the screen. (Demo not yet available)
