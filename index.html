LIBRARIES

Teemr

Teemr is a small JavaScript theme library that allows you to dynamically apply specific sets of css properties on your html elements based on JSON templates, thus, giving your web site or web app the capability to have multiple themes. Especially useful when managing color schemes. Try it below:


Speck

Speck is an HTML-based semi-customizable mini particle generator and animation. Spice up your website header with moving particles.

Workr

Workr is a JavaScript helper library that allows you to use the native Web Worker API more intuitively and creatively. Workr also enables its user to pass function references to a web worker instead of creating separate .js files -- effectively making any valid method from your codebase run on a separate thread.

TypeAnimation

TypeAnimation is a small JavaScript-powered text animation library that lets you animate certain portion(s) of text anywhere within your DOM and set customizable speed, indicator, and pauses.

Fancy Text Converter

Fancy Text Converter, just like how it's called, converts alphabetical fancy text into readable normal characters through a pre-defined set of characters in order to perform various tasks such as searching through strings that has fancy text on it.

Alertr

Alertr is a customizable JavaScript alert UI component similar to Windows 10 notifications. You can show stylized notification pop ups anywhere on the screen.
